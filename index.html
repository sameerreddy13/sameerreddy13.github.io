My Projects

Hurt Alert

Hurt Alert was a web application developed as a team, built with Python and Flask. It utilizes Berkeley crime data to present analytics and visualize this data in a heat map. More importantly it uses this data to route from point A to point B; finding the safest route to avoid areas of higher crime rates. Check out the application and repository

Text Editor

A text editor application built in Java with all basic features of a common text editor. Check out the repository.

Bear Maps

A mapping application for Berkeley built in Java as part of a university course. Generates a map of Berkeley with map navigation and shortest path routing from any two locations on the map.
